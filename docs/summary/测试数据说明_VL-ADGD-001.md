## 🎮 功能体验建议

### 1. 概览页面
- 查看3个平台的账户统计
- 查看机会分数分布(优秀/待改进/需关注)
- 查看4个优化案例的轮播展示
- 点击不同分数段卡片跳转到建议列表

### 2. 优化建议页面
- 查看4个分类的统计卡片
- 点击分类卡片筛选该分类的建议
- 测试搜索功能(输入账户ID或关键词)
- 使用筛选器(平台、分类、分数范围、状态)
- 点击建议查看详情
- 测试"采纳"和"忽略"操作

### 3. 账户管理页面
- 查看按平台分组的账户列表
- 展开各平台查看账户详情
- 测试添加新账户功能
- 查看账户的优化建议数和机会分数
- 查看账户余额和到期日期

## 📝 测试场景推荐

### 场景1: 紧急优化需求
1. 进入概览页,点击"需关注"卡片(3个账户)
2. 查看低分账户的建议列表
3. 重点关注"紧急"和"高优先级"建议
4. 采纳关键建议

### 场景2: 按分类优化
1. 进入优化建议页
2. 点击"预算"分类卡片
3. 查看所有预算相关建议
4. 筛选高影响分数的建议
5. 逐条处理建议

### 场景3: 多平台管理
1. 进入账户管理页
2. 查看Meta平台账户(6个)
3. 展开账户查看详情和统计
4. 切换到Google Ads平台(3个)
5. 对比不同平台的表现

### 场景4: 搜索和筛选
1. 在建议页面搜索"预算"
2. 筛选"META"平台
3. 筛选分数范围"0-40"
4. 查看筛选结果

## 🔧 数据重置

如果需要重置测试数据,请执行以下操作:

1. 打开浏览器开发者工具(F12)
2. 进入Application/Storage标签
3. 找到localStorage
4. 删除`vl_project_db`键
5. 刷新页面,数据将自动重新初始化
